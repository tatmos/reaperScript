# reaperScript

REAPER用のシンプルなスクリプト集です。MIDI編集やリージョン管理をちょっと便利にします。

## スクリプト一覧

- **tatAddLengthGrid.py**  
  ステップ入力したノートを、グリッド幅ぶんだけ伸ばします。  
  https://qiita.com/tatmos/items/ebb0241c34c1c83de7b7

- **tatLabel.py**  
  カンマ区切りの文字列で、複数リージョンを一括リネームします。  
  https://qiita.com/tatmos/items/f939e466897bd550be6c

- **tatNoteList.py**  
  選択したMIDIノート情報をテキスト化します。  
  https://qiita.com/tatmos/items/f5baccce480fe81aec09
